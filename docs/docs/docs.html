Getting Started

Everything you need to install and use the VATPHIL Updater to keep your EuroScope sector files up to date.

1

Installation

1
Download the installer
Head to the home page and click Download. The .exe file will download automatically.
2
Run the executable
Double-click VATPHIL-Updater.exe to launch the app. Windows may show a SmartScreen warning — click More info → Run anyway.
3
Set your installation directory
Click Browse Folder and select the folder where your sector files are located (e.g. Desktop\VATPHIL Sector).
Note: The updater creates a folder called VATPHIL-Updater
2

Using the Updater

1
Select the download location
Remember where you place it as it contains all the important files to get started.
2
After the files have been downloaded
Enter your VATSIM Password and ACARS Code.
3
Press Apply!
You can start controlling!
Done! The updater will show a green status when your installation matches the latest AIRAC data.
